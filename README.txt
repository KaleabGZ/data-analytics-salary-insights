# 💼 Job Market Analysis Project (2025)

This portfolio project analyzes **global data science related salaries** using **Python (Pandas, Seaborn, Matplotlib)** and **Tableau** to extract insights from roles across countries, company sizes, and experience levels.

---

## 📊 Project Overview

- 🎯 **Goal:** Analyze and visualize the salary trends of data professionals across regions and experience levels.
- 🧰 **Tools Used:**
  - Python (Pandas, Matplotlib, Seaborn, Plotly)
  - Tableau Public
  - Jupyter Notebook
  - Git & GitHub

---

## 🗂️ Project Structure

| File                          | Description                                |
|------------------------------|--------------------------------------------|
| `DataScience_salaries_2024.csv` | Raw dataset (from Kaggle / Open source)     |
| `cleaned_ds_jobs.csv`        | Cleaned dataset after preprocessing         |
| `data_cleaning_analysis.ipynb` | Jupyter Notebook with full data analysis   |
| `Job Market Visualization.twbx`      | Tableau Workbook (interactive dashboard)    |
| `README.md`                  | This project summary                        |

---

## 📈 Python Analysis (Jupyter Notebook)

The Python notebook includes:
- ✅ Data cleaning & preprocessing (nulls, dtypes, currency conversion)
- ✅ Exploratory Data Analysis (EDA)
- ✅ Visualizations:
  - Salary distribution by job title, experience, location
  - Top-paying countries & roles
  - Heatmaps, boxplots, bar charts

---

## 🌍 Tableau Dashboard

An **interactive dashboard** was created in Tableau to visualize:
- 🌐 Salary by Country (heatmap)
- 🧑‍💼 Salary by Experience Level (Entry, Mid, Senior)
- 🏢 Job type by Remote Work
- 📅 Salary distribution on different roles
